Projects

A non-exhaustive list of projects I have worked on:
ft_transcendence - Web-based Pong game
  • A single page application to play Pong, built using Fastify/Node.js backend together with SQLite as the database engine and vanilla Typescript plus Tailwind CSS frontend. All is built with microservices architecture in mind where the core features are segregated and decoupled into multiple smaller and individual components.
  • Comes with features similar to modern day applications such as user management (registration/account modification), live chatting and notifications (direct messages/game invitation), as well as accessibility improvements like server-side rendering for SEO.
Inception - Containerized WordPress application
  • A docker-focused project for building your own website using WordPress. The website can be spun up with just a single command and users are free to configure it to their liking.
  • The core application consists of WordPress, MariaDB, and NGINX as a reverse proxy. Several features are also included such as Adminer for database management, an FTP server for allowing remote access to application files, and Redis for site traffic caching.
miniRT - Ray-tracing from scratch
  • A declarative Ray-tracer, where scenes can be declared in a configuration file to be read and rendered by the program.
  • Written in C and utilizing a minimal graphical library from the 42 School, it implements recursive ray tracing as the main algorithm and Phong reflection model for the lighting to produce realistic images that have diffuse, specular, and ambient properties.
  • As ray tracing is expensive, the program employs the bounding volume hierarchy method to optimize the rendering process where it groups scene objects and efficiently calculates the color of each pixel.
